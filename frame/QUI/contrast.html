# QUI框架

介绍

乐高系统孵化出的前端开发框架，基于VUE，包括丰富的UI组件和Web工程体系，可快速搭建中、后台系统。

# QUI 组件

以上包括5个基础组件，17个表单组件，15个数据展示组件，5个反馈组件，6个导航组件，4个其他组件，另外还包括三个奇艺业务组件，CropUpload截图上传、Flowchart流程图、Linechart折线图。

# QUI 工程化体系

三套脚手架：单页面单模块、单页面多模块、多页面多模块

# 工程化体系介绍

SPPM单页面多模块工程代码结构：

模块分层的数据流：

Router(路由) -> Pages(页面) -> Components(组件) -> Store(通过发起action调用service) -> service(聚合dao和model的数据处理工厂) -> dao(对Fetch API的封装,进行接口调用)-> model(对数据进行二次处理) ->改变state，渲染页面

# 命令行工具

  • 项目脚手架
    • 用于快速搭建项目
    • SPSM（单页面单模块），SPMM（单页面多模块），MPMM（多页面多模块）分别适用于小型、中型、大型工程
  • 组件脚手架
    • component-scaffold,用于快速搭建组件开发原型
