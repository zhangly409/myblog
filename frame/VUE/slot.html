# Slot 插槽

# 定义

TIP

slot插槽是组件的一块HTML模板，这个模板显示不显示由父组件决定，显示的位置由子组件自身决定，slot写在组件的什么位置则显示在什么位置。

# 分类

  • 单个插槽
    = 默认插槽 = 匿名插槽
  • 具名插槽
    slot属性关联具名插槽，没有slot属性的html模板默认关联匿名插槽

TIP

slot是一块HTML模板，由父组件提供，子组件暴露出一个可以让父组件传入自定义内容的接口。
