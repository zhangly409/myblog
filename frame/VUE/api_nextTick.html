# Vue.nextTick

# 用法

在下次Dom更新循环结束后执行的延迟回调，修改数据后立即使用该方法，可获得更新后的dom.

# 事件循环说明

Vue在修改数据后视图不会立即更新，而是等同一事件循环的所有数据更新完成。

# 应用场景

需要在视图更新后，基于视图的操作
